# Tarsau

Sistem Programlama dersi için geliştirilmiş basit arşivleme programı.

## Derleme

```bash
make
````

## Arşiv Oluşturma

```bash
./tarsau -b t1 t2 t3.txt -o arsiv.sau
```

## Arşiv Açma

```bash
./tarsau -a arsiv.sau
```

veya

```bash
./tarsau -a arsiv.sau cikti_dizini
```

## Temizlik

```bash
make clean
```
